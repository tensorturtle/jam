# Jam Hugo Theme

Jam is a minimal personal blog and portfolio theme powered by Hugo. No JS!

### Features

- Minimalistic, clean and simple design
- Responsive design
- Inline CSS
- Atom feed
- Sass
- Seo optimized (Twitter cards, Facebook Open Graph, Schema.org)
- Ultra fast
